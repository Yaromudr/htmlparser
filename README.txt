Реализовать консольное приложение, которое ищет на странице https://news.yandex.ru/computers.html определенные слова
(указанные как параметр командной строки или взятые из файла) и выводит результат в виде пар слово-количество вхождений,
отсортированный по убыванию кол-ва вхождений.
Поиск д.б. регистронезависимым.
Падежи и т.д. не рассматриваются (т.е. слова крокодил, крокодилу, крокодилы являются разными словами).

Правила использования артефакта:
1) Собрать артефакт maven-ом
2) Найти в папке target/resources файл htmlparser.properties
3) Переложить файл рядом с файлом htmlparser-1.0-SNAPSHOT-jar-with-dependencies.jar
4) Запустить в консоли, указав слова для поиска в виде параметров, например: java -jar htmlparser-1.0-SNAPSHOT-jar-with-dependencies.jar США  интернет
